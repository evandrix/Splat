__call__ ( source,  event )

Accept event from an IEventSource (source)

This method should return True if the callback wishes to "consume" the event (i.e. prevent other callbacks from receiving it). However, the event source is free to ignore the callback's return value. The "consume an event" convention is normally used only by "stream" event sources such as distributors and semaphores, and by event sources with ongoing listeners/handlers (as opposed to one-time callbacks).
