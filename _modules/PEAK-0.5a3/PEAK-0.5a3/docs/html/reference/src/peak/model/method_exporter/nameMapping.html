Class: nameMapping ./src/peak/model/method_exporter.py

A mapping used for formatting outer method names

This class implements a mapping object with a few special tricks. Names to be looked up are split on ., and then each name part is looked up first as a method of the nameMapping object itself. If a method is found, it's called on the result of the previous lookup, or the default name if there was no previous lookup. If a method isn't found, the name part is looked up in the dictionary of the MethodExporter the nameMapping was created for.

See the documentation for individual method names to see what special names are available.
