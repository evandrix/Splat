processTable ( thisTable,  remainingTables )

Process thisTable, and optionally consume some remainingTables

remainingTables is an iterator over the remaining tables in the same document, if any. Any tables consumed from this iterator will be skipped by the calling document processor. This allows a table processor to take control of processing for subsequent tables in a document, e.g. when multiple tables are needed to process a single test.
